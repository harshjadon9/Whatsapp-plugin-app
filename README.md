## setup
```npm run build```

## running
```

# android emulator ( emulator configured )
expo start -a

# ios emulator
expo start -i

# physical device
expo start 
(scan QR from expo app)

# release
expo build:android
expo build:ios

```

sign up to expo and get auto signing :)
